## Run Locally

아래 명령어로 로컬에 직접 설치해보거나, 혹은 배포된 url로 가서 확인해볼 수 있습니다.  


Prerequisites:  Node.js  

**1. 로컬에 직접 설치**  

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`


**2. 임시 배포된 url**
